# A. 模拟信号转换


- 模拟到模拟转换 ( analog-to-analog conversion) 可以通过三种方法实现
	- 调幅 (amplitude modulation, AM)
	- 调频 (frequency modulation, FM) 
	- 调相 (phase modulation, PM)
