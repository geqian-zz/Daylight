# A. 协议



## A.1. 协议的三要素

- 语法 (syntax)，数据的结构或格式
- 语义 (semantics)，地址和控制信息
- 时序 (timing)，报文发送的时间和发送的速率



# B. 标准


- 事实上的标准，如 TCP/IP
- 法定的标准，如 OSI/RM
