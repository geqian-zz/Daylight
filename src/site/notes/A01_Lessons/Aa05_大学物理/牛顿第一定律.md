# A. 牛顿第一定律内容


## A.1. 描述定义

**合外力为 0 $\Leftrightarrow$ 质点处于平衡状态**


- **力** 是质点运动状态发生改变的原因
- **质量** 是物体惯性大小的量度
- **惯性** 是物体保持静止状态或匀速直线运动状态的性质
- [[A01_Lessons/Aa05_大学物理/惯性系\|惯性系]]



## A.2. 公式表达

 $$\sum \overrightarrow{F_i}=m \frac{\mathrm{d} \vec{v}}{\mathrm{~d} t}=0$$ 
 
 式中 $\sum \overrightarrow{F_i}$ 表示质点所受的合力



# B. 牛顿第一定律的意义


- 给出了质点的平衡条件
- 引进了惯性和力两个重要概念
