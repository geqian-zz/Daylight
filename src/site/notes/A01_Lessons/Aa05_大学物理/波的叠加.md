# A. 波的叠加原理

当几列机械波相遇时，相遇区域中任一点振动为各列波单独存在时在该点引起的振动位移的矢量和。相遇后，它们仍然保持各自原有的特征 (振幅、频率、波长、振动方向) 不变，并按原来的方向继续前进。
