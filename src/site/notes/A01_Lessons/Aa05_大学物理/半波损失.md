# A. 半波损失的定义

入射波在反射点振动相位与反射波在反射点振动相位相反的现象。


# B. 半波损失的条件


![Pasted image 20230509100516.png](/img/user/Z02_ObFiles/Attachments/Pasted%20image%2020230509100516.png)



- 波疏介质射向波密介值
- 只需要考虑反射情况,不需考虑折射情况
- 若反射时无能量损失，则入射波和反射波合成产生驻波。
