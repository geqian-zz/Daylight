# A. 位置矢量的定义

位移和位矢虽然都是矢量，但二者是两个不同的概念。


位矢指在某一时刻，**以坐标原点为起点**，以运动质点所在位置为终点的有向线段。

位移是在一段时间间隔内，从质点的起始位置引向质点的终止位置的有向线段。
