# A. 参考系的定义

- 为定性描述质点运动状态而选择的参考物体或物体系。
- 指研究物体运动时所选定的参照物体或彼此不作相对运动的物体系。


---

# B. 参照系选择的标准
参考系的选择是任意的，但应以观察方便和使运动的描述尽可能简单为原则，研究地面上物体的运动常选择地面为参考系。
