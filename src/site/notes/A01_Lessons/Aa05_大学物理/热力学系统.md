# A. 热力学系统的定义

热学中被确定为研究对象的物体，简称系统。系统以外的物体是外界。系统可分为

- 开放系统（系统与外界既交换物质也交换能量）
- 封闭系统（系统与外界不交换物质，只交换能量）
- 孤立系统（系统与外界既不交换物质，也不交换能量）
