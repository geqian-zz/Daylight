# A. 马吕斯定律的内容

人射线偏振光的光强为 $I_0$, 透过偏振片后, 透射光的光强 (不计偏振片对光的吸收) 为 $I$, 则
$$
I=I_0 \cos
{ #2}
 \alpha
$$
式中, $\alpha$ 为线偏振光的光矢量振动方向和偏振片偏振化方向之间的夹角。
