# A. 时间和空间

- 时间的含义：是指物质运动的持续性、顺序性
	- 特点：一维性，即一去不复返
- 空间的含义：是指物质运动的广延性、伸张性
	- 特点：三维性


## A.1. 时空与物质运动的关系

不可分割。时空是物质运动的时空，物质运动是时空中的物质运动。**时间和空间是运动着的物质的基本存在形式。**

物质运动总是在一定的时间和空间中进行的，没有离开物质运动的“纯粹”时间和空间，也没有离开时间和空间的物质运动。

物质、运动、时间、空间具有内在的统一性。从根本上说，世界上除了运动的物质，什么也没有。
