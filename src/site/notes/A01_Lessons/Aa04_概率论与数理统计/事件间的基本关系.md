# A. 事件间基本关系

| 关系名称 | 符号表示 |
| :--: | :--: |
| 和事件 | $A \cup B(A+B)$ |
| 积事件 | $A \cap B(AB)$ |
| 差事件 | $A-B$ |
| 包含关系 | $A \subset B$ |
| 互斥/互不相容事件 | $A \cap B=\emptyset$ |
| 对立事件 | $A \cap B=\emptyset$ 且 $A \cup B=\Omega$ |
| 独立事件 | $P(A B)=P(A) \times P(B)$ |

- 对立事件一定是互斥事件


## A.1. 差事件的运算规律

$$
P(A-B)=P(A\bar{B})=P(A-AB)=P(A)-P(AB)
$$
