# A. 信号频谱的定义


- 信号的某种特征量随信号频率变化的关系，称为信号的频谱，所画出的图形称为信号的频谱图
- 周期信号的频谱是指周期信号中各次谐波幅值、相位随频率的变化关系



- 频谱图
	- 单边频谱图 -> 三角形式
	- 双边频谱图 -> 虚指数形式
