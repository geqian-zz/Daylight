# A. 零点漂移的定义

$$U_o = AU * U_i$$



放大电路在无输入信号的情况下，输出电压 uo 却出现缓慢、不规则波动的现象。



# B. 产生零点漂移的原因


产生零点漂移的主要原因是温度的影响
所以有时也用温度漂移或时间漂移来表示

零漂的大小，是通过比较折算到输入的值来确定的
