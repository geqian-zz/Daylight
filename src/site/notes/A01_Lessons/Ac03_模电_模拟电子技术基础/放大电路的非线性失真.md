# A. 非线性失真

## A.1. Q 点过低 -> 截止失真


- NPN 管： 顶部失真为截止失真
- PNP 管： 底部失真为截止失真


## A.2. Q 点过高 -> 饱和失真


- NPN 管：底部失真为饱和失真
- PNP 管：顶部失真为饱和失真
